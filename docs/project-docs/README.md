## 适用范围

HR Agent 是一个面向企业 HR 场景的智能人力资源管理系统，当前项目覆盖：

- HR Agent 对话入口与受控工具编排
- JD 生成、评分标准生成、简历筛选、面试方案生成
- 试卷生成、考试管理与结果查看
- 文档上传、知识库管理、知识问答
- 用户登录、角色管理、邮箱配置、统计看板

## 编写口径

- 以当前 FastAPI 后端、Vue 3 前端、PostgreSQL + pgvector、Dify 工作流、LLM/RAG 服务为依据。
- 不将系统描述为完整 ATS、HRIS 或全自动招聘决策平台。
- 将 CI/CD、集中监控、服务发现等平台能力作为上线前置或运维协作项，不默认视为本项目代码已内建能力。
